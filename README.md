# Dividitto

An app to register your home expenses and divided based on family member income

## Dev
Start your enviroment running:
    
```bash
pipenv install
```

Configure db running:

```bash
make db_configure
```

Then start db running:

```bash
make db_start
```

The run migrations:

```bash
make migrate
```

Finally run the app:

```bash
make run
```
